# CIS-320-Wind-Turbine

This is a model smart wind turbine to demonstrate the functionality and design of modern wind turbines. This project utilizes an Arduino Mega 2560 and many components ([Components]()) to successfully autonmate normal functions. One unique addition to the turbine that is not currently in most wind turbines is the idea of a radar to help protect the structure from approaching danger.

For more information check out the [Wiki](https://github.com/robert-csu/CIS-320-Wind-Turbine/wiki)
